About ForgeXplorer

ForgeXplorer is a web-based digital image forensic system that detects image forgery using Compression Error Analysis (CEA) and a Convolutional Neural Network (CNN).

Project Objective

To automatically identify whether a digital image is Authentic or Forged by analyzing compression inconsistencies introduced by image manipulation techniques such as copy-move and splicing.

Technologies Used

  • Compression Error Analysis (CEA)
  • Convolutional Neural Network (CNN)

⚙️ How the System Works

  1. User uploads a digital image.
  2. Image undergoes Compression Error Analysis (CEA).
  3. CNN extracts forensic features.
  4. Model predicts Authentic or Forged.
  5. Confidence score is generated.

📊 Model Performance Comparison

Two deep learning models were evaluated for detecting forged images: a custom CNN and a VGG16 transfer learning model. The CNN achieved higher accuracy because it was specifically trained on Compression Error Analysis (CEA) images used in the system.

Model Architecture Accuracy Remarks
